Age of Ascent

Unity - C#, .NET Core

Age of Ascent is a new space co-op and combat game featuring direct-piloting solo and co-op missions and real-time PvE and PvP battles set in stunning space scenes.

I worked across all areas of the game, designing and implementing core game systems such as AI enemies, in game objective systems and tools to help other developers. I worked heavily on the networking of the game. Using .NET core in the backend and Unity with C# for the frontend. Developing systems to load player data from our database. I refactored many areas of the networking system: ensuring AI systems were networked and fixing numerous existing bugs with the network stack.
