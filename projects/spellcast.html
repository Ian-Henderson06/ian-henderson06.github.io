Spellcast

Unity - C#

Spellcast is a 1v1 online multiplayer wizard fighting game. Each player has a selection of spell elements. Players must use opposing elements to block one another.

I created the game myself. Spellcast is fully server authoritative, and uses numerous modern techniques to ensure a smooth experience. Spellcast also supports Steam multiplayer.
